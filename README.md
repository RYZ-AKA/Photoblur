# PHOTO BOOTH 📸

A vintage newspaper-inspired photobooth experience built with pure HTML, CSS, and JavaScript.

Users can capture photos using a ✌ peace hand gesture, automatically generating aesthetic polaroid memories surrounded by floral decorations.

## Features

- 📸 Camera access
- ✌ Peace gesture detection using MediaPipe Hands
- 🌫 Capture blur animation
- 🖼 Automatic polaroid generation
- 🌷 Floral scrapbook decorations
- 📰 Vintage newspaper aesthetic UI
- 💾 Download final composition as image
- 📱 Mobile-friendly responsive design

## Tech Stack

- HTML5
- CSS3
- JavaScript
- MediaPipe Hands
- html2canvas

## How It Works

1. Open the website.
2. Enter the photobooth.
3. Allow camera access.
4. Make a ✌ peace sign.
5. The camera captures automatically.
6. Repeat for the second photo.
7. Download your finished polaroid memory.
